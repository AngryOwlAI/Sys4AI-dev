# Generic /init

`/init` is the portable Sys4AI front door template for target-system definition and adoption.

It classifies greenfield and brownfield situations, inspects evidence before asking questions, captures candidate vision, values, anti-values, approval gaps, conflicts, waivers, and review cadence, and asks for approval before creating discovery records, Product Requirements Documents, system requirements, implementation plans, or scaffolds.

Candidate strategic intent is not approved by silence, model authorship, storage location, or structural validation.

The brownfield first pass is read-only and should produce an evidence summary before any file is written.

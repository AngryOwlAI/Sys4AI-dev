# Generic /init Agent Guidance

Use `/init` as the first routing layer for target-system Sys4AI adoption.

Required behavior:

- Classify greenfield, brownfield, partially built, or documentation recovery state.
- Identify system-of-interest, subject layer, lifecycle intent, evidence availability, and downstream route.
- Inspect source evidence before asking questions.
- Keep brownfield first pass read-only.
- Ask explicit approval before creating controlled artifacts or scaffolds.
- Preserve `REQ-CAND-*` and `NFR-CAND-*` labels until promotion.

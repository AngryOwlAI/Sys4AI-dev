# sys-for-ai

`sys-for-ai` is a domain-agnostic system development and management framework for AI agents.

Its purpose is meta-agentic: a root AI agent, such as Codex, uses `sys-for-ai` to design, develop, run, improve, and maintain target agentic systems for specific use cases.

Phase 0 defines the product and system-design baseline: roles, artifact contracts, traceability rules, control-loop requirements, AgentJob-style bounded work, source-first memory, and source/version-control governance.

Implementation initialization belongs to Phase 1 and should be defined later in its own artifact.

# Init

`/init` is the Sys4AI-dev runtime front door for system definition and adoption.

It classifies the situation, inspects available evidence, summarizes the route, and asks for approval before writing discovery records, Product Requirements Documents, implementation plans, AgentJobs, or scaffolds.

The brownfield first pass is inspection-only. It may summarize repository evidence, but it must not modify the repository until the user or Director approves the next controlled artifact.

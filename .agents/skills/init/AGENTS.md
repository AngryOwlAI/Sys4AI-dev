# Init Agent Guidance

Use this skill as the first routing layer for Sys4AI adoption.

Required discipline:

- Inspect available evidence before asking questions.
- Preserve `Sys4AI-dev`, `Sys4AI/`, `.agents/skills`, `.codex/skills`, and `Sys4AI/skills/core` boundaries.
- Treat brownfield first-pass work as read-only.
- Ask explicit approval before creating controlled artifacts or mutating a repository.
- Preserve candidate requirement labels as `REQ-CAND-*` or `NFR-CAND-*`.
- Route to existing downstream skills instead of duplicating their full behavior.
